# 開発環境

このリアルタイムマップはVueで作られています。追加・修正大歓迎です。

## クローン
```
git clone https://github.com/yu-ichiro/sos-realtime-map
```

## セットアップ
```
yarn install
```

### 開発用ホットリロード付きサーバーの立ち上げ
```
yarn serve
```

### 書式修正
```
yarn lint
```
